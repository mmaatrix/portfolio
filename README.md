# Professional Portfolio Website

This repository contains my personal portfolio website, built to demonstrate core web development skills using HTML, CSS, and JavaScript. The project showcases a clean structure, accessible design, responsive layout, and interactive features, and is deployed using GitHub Pages.

## Project Overview
This portfolio presents my professional background, technical expertise, and selected projects. It was developed as part of a web development project and follows best practices for modern, accessible, and responsive web design.

## Features
- Semantic HTML structure for clarity and accessibility
- Accessibility enhancements including alt text and ARIA attributes
- Responsive layout using CSS Flexbox and Grid
- Consistent styling applied across the site
- JavaScript-driven interactivity:
  - Dark mode toggle
  - Project filtering
  - Back-to-top button
  - Mobile navigation menu
  - Contact form validation

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)
- GitHub Pages for deployment

## Accessibility & Responsiveness
The website is designed with accessibility in mind, including:
- Descriptive alt attributes for images
- Keyboard-accessible navigation
- Proper heading hierarchy
- Responsive behavior across desktop, tablet, and mobile devices

## Deployment
The project is hosted on GitHub Pages.

To deploy:
1. Upload project files to a GitHub repository
2. Commit changes to the `main` branch
3. Enable GitHub Pages from repository settings
4. Select the `main` branch as the deployment source
